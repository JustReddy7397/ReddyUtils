# ReddyUtils [![](https://jitpack.io/v/JustReddy7397/ReddyUtils.svg)](https://jitpack.io/#JustReddy7397/ReddyUtils)
Hey there! With this Utils API you can easily create yml files, commands with arguments ( /test hello for example ), add colors to your messages / lists, get a players name by UUID or get a player by UUID and create a scoreboard!

**NOTE:** You are not allowed to fork this!

[Documentation](https://wiki.justreddy.ga)
